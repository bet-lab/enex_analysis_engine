\begin{abstract}
This document presents the mathematical formulations implemented in the \texttt{enex\_analysis\_engine} Python package.
Each section covers one simulation module, specifying the governing equations, modelling assumptions, and literature references.
Equations are numbered consistently with the source code variable names to facilitate cross-referencing.
\end{abstract}
