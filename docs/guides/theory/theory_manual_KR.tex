\begin{abstract}
본 문서는 \texttt{enex\_analysis\_engine} Python 패키지에 구현된 수학적 정식화를 제시한다.
각 절은 하나의 시뮬레이션 모듈을 다루며, 지배방정식, 모델링 가정, 참고문헌을 명시한다.
수식 번호는 소스코드의 변수명과 일관되게 부여하여 상호 참조가 가능하도록 하였다.
\end{abstract}
