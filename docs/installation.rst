Installation
=============

Installing the Package
----------------------

To use the models, you need to install the required packages. Run the following commands:

Using uv (recommended)
^^^^^^^^^^^^^^^^^^^^^^^

.. code-block:: bash

   uv add git+https://github.com/BET-lab/En_system_Ex_analysis.git

.. code-block:: bash

   uv sync

Using pip
^^^^^^^^^

.. code-block:: bash

   pip install git+https://github.com/BET-lab/En_system_Ex_analysis.git

Dependencies
------------

The package requires the following dependencies:

* dartwork-mpl
* ipykernel (>=6.29.5)
* matplotlib (>=3.10.0)
* notebook (>=7.3.2)
* numpy (>=2.2.3)
* pandas (>=2.2.3)
* salib (>=1.5.1)
* scipy (>=1.15.2)
* seaborn (>=0.13.2)
* tqdm (>=4.67.1)

These will be automatically installed when you install the package.
